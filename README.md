# 🏗️ Features

The Goal of this project is to implement all features of discord RPC protocol.

- [x] Handshake (Faked for now)
- [x] Bridge Server
- [x] IPC Server (Partial)
- [ ] Websocket Server
- [ ] Process Detection
- [ ] All Commands
- [ ] Systemd Deamon
- [ ] Windows Support
